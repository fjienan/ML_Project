\section{数据预处理}

\label{sec:preprocessing}
\subsection{数据概述}
本项目使用乳腺癌蛋白质表达与临床数据，共包含三个数据集：
\begin{enumerate}
    \item \textbf{77\_cancer\_proteomes\_CPTAC\_itraq.csv}：77例乳腺癌样本的蛋白质表达数据，含12000余种蛋白质的log2表达值。
    \item \textbf{clinical\_data\_breast\_cancer.csv}：临床信息数据，包含样本PAM50分子分型标签。
    \item \textbf{PAM50\_proteins.csv}：PAM50分型系统对应的特征蛋白列表。
\end{enumerate}
\subsection{数据预处理流程}
\subsubsection{数据清洗与格式标准化}
\begin{enumerate}
    \item 删除非数值特征列 \texttt{gene\_symbol}、\texttt{gene\_name}。
    \item 去除样本ID中的多余符号，统一格式化为 \texttt{TCGA-XXX}。
    \item 删除重复样本列，保留77例有效癌症样本。
    \item 将矩阵转置，使每行代表一个样本，每列代表一个蛋白质特征。
\end{enumerate}

\subsubsection{数据匹配与特征筛选}
\begin{enumerate}
    \item 根据样本ID将蛋白质表达数据与临床数据进行匹配。
    \item 依据PAM50蛋白列表，仅保留与分型相关的蛋白质特征。
\end{enumerate}

\subsubsection{缺失值处理}
采用中位数填充缺失值，避免样本丢失并减少异常值影响：
\[
x_{\text{imputed}} = \text{median}(x_{\text{observed}})
\]

\subsubsection{训练集与测试集划分}
按照项目要求，采用分层抽样划分：
\begin{itemize}
    \item 训练集：43例（含完整PAM50分型）
    \item 测试集：34例
    \item 分层依据：\texttt{PAM50 mRNA}，保证分布一致
\end{itemize}

\subsection{数据处理结论}
\begin{enumerate}
    \item 完成重复列删除、样本ID标准化、矩阵转置等清洗步骤。
    \item 成功匹配蛋白质表达数据与临床标签数据。
    \item 中位数填充有效保留数据分布，无明显偏差。
    \item 分层抽样保证训练集与测试集PAM50分型分布一致。
    \item 特征筛选降低维度，数据可直接用于聚类与分类建模。
\end{enumerate}
