\section{数据预处理操作}
\label{sec:preprocessing}

\subsection{数据概述}
本项目使用乳腺癌蛋白质表达与临床数据，共包含三个数据集：
\begin{enumerate}
    \item \textbf{77\_cancer\_proteomes\_CPTAC\_itraq.csv}：77例乳腺癌样本的蛋白质表达数据，含12000余种蛋白质的log2表达值。
    \item \textbf{clinical\_data\_breast\_cancer.csv}：临床信息数据，包含样本PAM50分子分型标签。
    \item \textbf{PAM50\_proteins.csv}：PAM50分型系统对应的特征蛋白列表。
\end{enumerate}

\subsection{基础数据清洗}
\begin{enumerate}
    \item 删除非数值特征列 \texttt{gene\_symbol}、\texttt{gene\_name}。
    \item 去除样本ID多余符号，统一规范为 \texttt{TCGA-XXX} 格式。
    \item 剔除重复样本列，保留77份有效癌症样本数据。
    \item 矩阵行列转置，设定单行对应单个样本、单列对应单项蛋白特征。
\end{enumerate}

\subsection{特征匹配与筛选}
\begin{enumerate}
    \item 依据样本编号关联蛋白质表达数据与临床分型数据。
    \item 参照PAM50特征蛋白清单，仅保留分型关联蛋白特征。
\end{enumerate}

\subsection{缺失值与异常值处理}
由于蛋白质表达数据存在偏态分布与异常值，均值易受极端值干扰，
而中位数具有稳健性，能更合理地代表蛋白质表达的中心趋势,
因此，我们采用中位数填充方式补齐缺失数据，规避样本丢失问题，弱化异常数据干扰：
\[
x_{\text{imputed}} = \text{median}(x_{\text{observed}})
\]
本数据集的极端数值属于正常的蛋白表达差异，并非噪声错误，
且中位数填充与标准化已有效降低异常值干扰，因此本实验不进行异常值截断，
保留原始数据的生物学分布特征。

\subsection{数据标准化}
由于不同蛋白质的表达值存在天然的量纲与数值尺度差异，
未标准化数据会导致尺度较大的特征在距离计算中占据主导，
干扰聚类与分类模型的训练效果。
因此，本研究在建模前对所有蛋白特征进行标准化处理，
将所有特征统一为均值为 0、方差为 1 的标准分布，
消除特征尺度不一致带来的偏差，
保证各蛋白质特征对模型的贡献权重公平、有效，
提升后续聚类与分类结果的准确性与可靠性。

\subsection{数据集划分}
采用分层抽样方式拆分样本，保证分型分布均衡统一：
\begin{itemize}
    \item 训练集：共计43例，携带完整PAM50分型标签
    \item 测试集：共计34例
    \item 分层依据：\texttt{PAM50 mRNA} 亚型类别
\end{itemize}
